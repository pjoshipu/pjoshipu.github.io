🦞 OpenClaw Personal Agent

Kitty — Personal AI Agent

An always-on personal AI assistant built with OpenClaw, running across multiple channels from a self-hosted gateway. Powered by Claude (Anthropic), Kitty handles conversations, reminders, and tasks — available on WebChat and Telegram.

OpenClaw acts as a local-first gateway that routes messages from any connected channel to the Claude AI model, maintains session context, and delivers responses back — all from a single self-hosted instance.
