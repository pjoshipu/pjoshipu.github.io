⚙️ Agentic AI in Platform Engineering

Multi-Agent AI System for Enterprise Platform Operations

A comprehensive platform showcasing 5 specialized AI agents built with Claude 3 Haiku and Python. These agents solve critical platform engineering challenges: CI/CD diagnostics, release readiness evaluation, multi-agent orchestration, developer onboarding, and personalized AI assistance. Includes interactive demos with architecture diagrams, workflow visualizations, and real-time agent interactions.

The 5 AI Agents

🔍
Workflow Diagnostic Agent
Automatically analyzes failed CI/CD pipelines and provides root cause analysis with actionable fixes.
  • Log analysis and pattern recognition
  • Root cause identification
  • Automated fix suggestions
  • GitHub Actions integration
✅
Release Readiness Agent
Evaluates quality gates across multiple metrics to make intelligent release/rollback decisions.
  • Multi-metric quality assessment
  • Risk-based decision making
  • Progressive delivery strategies
  • Automated rollback triggers
🤝
Multi-Agent Orchestration
Specialized agents coordinate to handle cost optimization and incident response workflows.
  • Agent coordination patterns
  • Cost vs. performance trade-offs
  • Incident triage and response
  • Observability integration
👨‍💻
Developer Portal Agent
Provides AI-powered developer onboarding with personalized guidance and service maturity insights.
  • Personalized recommendations
  • Service maturity assessment
  • Golden path templates
  • Developer productivity metrics
🤖
AI Copilot
Your personal AI assistant for platform questions and development guidance with context-aware help.
  • Context-aware assistance
  • Experience-tailored guidance
  • Real-time troubleshooting
  • Best practices suggestions

Getting Started

Local Development

cd agentic-ai-platformengineering
npm install
npm run dev

# App runs on http://localhost:8080/platform/

Set API Key

# Add to .env file:
VITE_ANTHROPIC_API_KEY=sk-ant-...

Deploy to GitHub Pages

npm run build
npm run deploy
